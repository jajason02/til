# 비트마스킹 in dfs

> 2026-04-18

## 배운 것

### 1. SET
```python
visited = visited | (1 << x)
```
- 목적: x번째 위치가 방문했음을 표시
- 동작: OR연산으로 해당 비트를 1로 만듦

> DFS/백트래킹에서 "이 노드를 방문했다"고 표시

### 2. CHECK
```python
if ((visited & (1 << x)) != 0)
```
- 목적: x번째 위치가 방문했는지 확인
- 동작: AND연산으로 해당 비트가 1인지 체크

> 이미 방문했는지 방문 여부를 확인

### 3. CLEAR
```python
visited = visited & (~(1 << x))
```
- 목적: x번째 위치의 방문 표시를 해제
- 동작: NOT으로 비트를 반전 -> AND로 해당 위치만 0으로 만듦

> 백트래킹 시 미방문으로 돌려놓을 때 사용


## 막혔던 것 / 해결

1. 비트마스킹으로 만드는 부분집합 개념과 여기서 방문처리하는 것이 뭔가 로직이 햇갈렸는데 이번에 좀 구분을 할 수 있을 것 같다.

2. 비트마스킹 부분 너무 어렵고 개념이 많다고 생각해서 암담했는데 저 3가지 외에는 알고리즘 문제 풀이에 잘 쓰이지 않는다고 하니 저 3개부터 잘 숙달해봐야겠다.
