# 신경망 / 손실함수 / 경사하강법 / 로지스틱 회귀

> 2025-04-16

## 배운 것

### RSS (잔차제곱합)

```
RSS = Σ(yi - ŷi)²
```

학습 목적 = RSS를 최소화하는 파라미터(가중치) 찾기.

### 신경망 구조

- **입력층**: Feature 입력
- **은닉층**: 복잡한 특징 추출. 층이 깊을수록 Deep Learning.
- **출력층**: 최종 예측값 출력
- 각 연결선 = 가중치(Weight). 입력×가중치 + 편향(Bias) → 활성화 함수 → 다음 층.

### 손실함수

| 손실함수 | 주로 사용 |
|:---|:---|
| MSE | 회귀 |
| Cross-Entropy | 분류 (로지스틱 회귀 등) |

### 경사하강법 (Gradient Descent)

```
φ_j := φ_j - α × (∂Loss / ∂φ_j)
```

- α (학습률): 너무 크면 발산(오버슈팅), 너무 작으면 학습 느림
- 기울기가 낮아지는 방향으로 가중치를 조금씩 이동

### 로지스틱 회귀

이름은 회귀지만 실제로는 **분류** 알고리즘.

**시그모이드 함수**: 어떤 수든 0~1 사이로 압축.

```
σ(z) = 1 / (1 + e^(-z))
```

- 확률 ≥ 0.5 → 1 (Positive)
- 확률 < 0.5 → 0 (Negative)
- 손실함수: MSE 대신 **Binary Cross-Entropy** 사용.

## 막혔던 것 / 해결

로지스틱 회귀에서 왜 MSE를 안 쓰는지 궁금했음.  
시그모이드 출력에 MSE를 적용하면 손실 함수가 non-convex가 되어 경사하강법이 제대로 작동 안 함.  
Cross-Entropy는 볼록(convex)해서 최솟값을 잘 찾음.
