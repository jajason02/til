# AI / ML / DL 개요

> 2026-03

## 배운 것

### 정의

- **AI**: 인지·학습·추론을 통해 목표를 달성하는 시스템
- **ML**: AI 안에서 데이터로부터 규칙을 학습하는 방법론
- **DL**: ML 안에서 신경망(Neural Network)을 사용하는 방법론

```
AI ⊃ ML ⊃ DL
```

### 핵심 개념

- **Feature(피처)**: 모델이 예측에 사용하는 입력값. 예측의 근거.
- **Label(라벨)**: 모델이 맞춰야 하는 정답값. 학습의 목표.

**예시 — 스팸 분류**
- Feature: 메일 제목, 발신자, 단어 빈도
- Label: 스팸 / 정상

### 학습이란

가설공간(모든 후보 함수 집합)에서 손실 함수를 최소화하는 함수를 찾는 과정.

1. 함수 하나 선택
2. 전체 데이터로 예측
3. 손실 함수로 오차 계산
4. 파라미터 조정
5. 반복

### 왜 f()를 학습하는가

- **예측**: 새 입력에 대한 출력 추정
- **중요 특성 파악**: 어떤 피처가 결과에 영향을 주는지
- **해석 가능성**: 피처와 결과의 관계 이해

## 막혔던 것 / 해결

AI/ML/DL이 혼용돼서 헷갈렸는데, 포함 관계(AI ⊃ ML ⊃ DL)로 정리하니 명확해짐.  
ML은 방법론, DL은 그 안에서 신경망을 쓰는 특정 방법.
