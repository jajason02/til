# CNN / 합성곱 연산 / 주요 모델

> 2025-04-16

## 배운 것

### CNN 핵심 동기

완전 연결층(FC)은 이미지를 1D로 펼치는 과정에서 **공간 구조 정보가 손실**.  
CNN은 필터(커널)로 국소 특징을 유지하며 추출.  
학습 대상 = **필터의 값**.

### 합성곱 연산

입력 위를 필터가 슬라이딩하며 대응 원소 곱의 합 → Feature Map 생성.

**출력 크기 공식:**
```
Output Size = (H + 2P - F) / S + 1

H: 입력 크기, F: 필터 크기, P: 패딩, S: 스트라이드

예) H=32, F=5, P=0, S=1
→ (32 + 0 - 5) / 1 + 1 = 28
```

### Stride & Padding

- **Stride**: 필터 이동 보폭. 클수록 출력 크기 작아짐.
- **Padding**: 외곽에 0 채움. 출력 크기 유지 + 가장자리 정보 보존.

### Pooling

파라미터 없이 차원 축소.
- Max Pooling: 영역 내 최댓값 선택 (가장 많이 사용)
- Average Pooling: 영역 내 평균

### 주요 CNN 모델

| 모델 | 핵심 | 의의 |
|:---|:---|:---|
| AlexNet (2012) | GPU + ReLU + Dropout | 딥러닝 시대 개막 |
| VGGNet (2014) | 3×3 필터만 여러 개 쌓기 | "깊게 쌓는 게 답" 정착 |
| ResNet (2015) | Skip Connection, 잔차 학습 | 기울기 소실 해결, 152층 가능 |
| MobileNet (2017) | Depthwise Separable Conv | 연산량 1/9, 온디바이스 AI |

VGGNet에서 3×3 두 번 = 5×5 한 번의 효과이지만, 파라미터는 적고 비선형성은 더 많음.

## 막혔던 것 / 해결

ResNet의 Skip Connection이 왜 기울기 소실을 해결하는지 이해 안 됐음.  
역전파 시 기울기가 Skip Connection 경로를 통해 그대로 전달되어 곱셈이 누적되지 않음.  
→ 깊은 층에도 기울기가 잘 전달됨.
