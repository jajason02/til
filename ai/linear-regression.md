# 선형 회귀 (Linear Regression)

> 2025-04-16

## 배운 것

입력 변수(X)와 출력 변수(Y) 사이의 관계를 직선으로 근사하는 통계적 방법.

### 단순 선형 회귀

```
Y = β0 + β1X + ε
```

- β0: 절편
- β1: 기울기  
- ε: 관측 오차 (E(ε) = 0)
- Ŷ (hat 표기): 추정값

### 최소제곱법 (Least Squares)

실제값과 예측값의 차이(잔차)를 제곱해서 합한 RSS(잔차제곱합)를 최소화하는 β0, β1을 찾는 방법.

```
RSS = Σ(yi - ŷi)²  →  최소화
```

### 데이터 = 참 함수 + 오차

- 미지의 함수 f: feature와 label 사이의 실제 평균 관계. 직접 관측 불가.
- 오차 ε: 측정 기기 한계, 환경 요인 등으로 섞이는 노이즈.
- 우리가 보는 데이터는 항상 `f(x) + ε` 형태.

## 막혔던 것 / 해결

"데이터에서 직접 f를 알 수 없다"는 게 처음엔 추상적으로 느껴졌음.  
오차가 포함된 점(데이터)만 관측 가능하고, f는 그걸 바탕으로 추정하는 것임을 이해함.
