# 파운데이션 모델 / CLIP / VLM / 이미지 생성

> 2025-04-16

## 배운 것

### 파운데이션 모델

특정 작업에 국한되지 않고 다양한 하위 작업에 범용 적용 가능한 대규모 모델.

| 특징 | 의미 |
|:---|:---|
| 대규모(Scale) | 수천억 파라미터, 전 지구적 규모 데이터 |
| 적응성(Adaptability) | 적은 데이터로 Fine-tuning → 특화 모델로 변신 |
| 범용성(Generality) | 텍스트·이미지·음성 등 멀티모달 처리 |

일정 규모를 넘으면 학습 안 한 능력이 갑자기 나타남 → **창발(Emergent Abilities)**.

### CLIP

이미지 인코더 + 텍스트 인코더로 (이미지, 텍스트) 쌍을 같은 벡터 공간에 매핑.

**대조 학습(Contrastive Learning):**
- 맞는 쌍(이미지↔설명)의 내적 → 최대화
- 틀린 쌍의 내적 → 최소화

**Zero-shot 분류**: 한 번도 보지 못한 카테고리도 텍스트 후보와 유사도 비교로 맞힘.

**SigLIP**: CLIP의 Softmax → **Sigmoid 손실함수**로 변경. 더 효율적이고 안정적.

### VLM (Vision-Language Model)

이미지 + 텍스트를 동시에 이해하는 모델. 학습 2단계:

1. **정렬 학습**: 이미지 인코더·LLM 고정, **커넥터만** 학습 → "이 이미지 = 이 단어"
2. **지시 미세 조정**: (이미지, 질문, 답변) 데이터로 LLM도 함께 수정 → 복잡한 명령 수행

이미지 정보 전달 방식:
- **직접 주입(Projection)**: 이미지 패치 전부 LLM에. 정보 손실 적지만 무거움. (LLaVA)
- **압축 전달(Q-Former)**: 중요한 정보만 추려서 전달. (BLIP-2)

### 이미지 생성 모델

| 모델 | 핵심 |
|:---|:---|
| Stable Diffusion | 오픈소스 확산 모델 |
| DALL-E 3 | 긴 프롬프트 정확 구현 |
| FLUX.1 | 오픈 가중치 끝판왕, 사실적 인체 묘사 |

최신 트렌드: U-Net → **DiT(Diffusion Transformer)** 구조로 전환 → 성능 폭발.

## 막혔던 것 / 해결

VLM에서 커넥터가 하는 역할이 뭔지 헷갈렸음.  
이미지 인코더 출력(Feature Map)과 LLM이 기대하는 입력(텍스트 임베딩) 형태가 달라서 변환이 필요.  
커넥터가 이미지 특징을 LLM이 "단어처럼" 인식할 수 있게 다듬어줌.
