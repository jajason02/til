## 코사인 유사도

```
A=[1,2,3], B=[2,1,3]

A·B = 1×2 + 2×1 + 3×3 = 13
|A| = √(1+4+9) = √14
|B| = √(4+1+9) = √14

유사도 = 13 / 14 ≈ 0.929
```

## TF-IDF

```
문서 100개 중 10개에 "AI" 등장, 특정 문서에서 5번(전체 50단어)

TF = 5/50 = 0.1
IDF = log(100/10) = log(10) ≈ 2.303
TF-IDF = 0.1 × 2.303 ≈ 0.230
```

## 엔트로피

```
H = -Σ p_k × log₂(p_k)

예) [0.7, 0.2, 0.1]
H = -(0.7×(-0.515) + 0.2×(-2.322) + 0.1×(-3.322))
  = 1.157 bits

완전 확신 [1.0, 0, 0] → H = 0 (가장 좋은 상태)
엔트로피 높을수록 불확실 → TTA에서 걸러내야 할 샘플
```

## Attention 메모리

```
시퀀스 길이 1024, 32비트:
1024 × 1024 × 4 bytes = 4MB

시퀀스가 2배(2048)면:
2048 × 2048 × 4 = 16MB (4배 증가)
→ Attention은 시퀀스 길이의 제곱에 비례
```

## 이진수 ↔ 십진수

```
이진수 1011.101 → 십진수:
정수부: 8+0+2+1 = 11
소수부: 0.5+0+0.125 = 0.625
→ 11.625

십진수 6.75 → 이진수:
정수부 6: 6÷2=3r0, 3÷2=1r1, 1÷2=0r1 → 110
소수부 0.75: 0.75×2=1.5→1, 0.5×2=1.0→1 → .11
→ 110.11
```

## IEEE 754 변환

```
십진수 -6.75 → IEEE 754 단정밀도:

1. 이진수: 6.75 = 110.11
2. 정규화: 1.1011 × 2²
3. S = 1 (음수)
4. Exponent = 2+127 = 129 = 10000001
5. Fraction = 10110000000000000000000

결과: 1 10000001 10110000000000000000000

---

0 10000010 10100000000000000000000 → 십진수:
S=0(양수), Exp=130-127=3, Fraction=1.101
1.101 × 2³ = 1101 = 13.0
```

## 2의 보수

```
-45를 8비트로:
45 = 00101101
반전: 11010010
+1:  11010011

4비트에서 7+2:
0111 + 0010 = 1001 → 2의 보수로 -7
양수+양수=음수 → 오버플로우! (4비트 범위: -8~+7)
```

## 고정 소수점 연산

```
4.4 고정소수점에서 1.5 × 0.5:

1.5 → 내부값 24
0.5 → 내부값 8
24 × 8 = 192

192 >> 4 = 12 → 소수점: 0.75 ✓
```

## 양자화 계산 (S, Z)

```
범위 [-1.0, 1.0]을 8비트(0~255)로:

S = 2.0 / 255 ≈ 0.00784
Z = floor(1.0 / 0.00784) = 127

실수 0.5 양자화:
q = round(0.5 / 0.00784 + 127) = 191

역양자화:
r = 0.00784 × (191 - 127) ≈ 0.502
```

## 프루닝 메모리 계산

```
원본: 1,000,000개 × 32비트 = 32,000,000비트

70% 프루닝:
300,000개 × 32비트 = 9,600,000비트

4비트 양자화:
1,000,000개 × 4비트 = 4,000,000비트

→ 70% 프루닝 ≈ 약 9.6비트로 줄인 효과
→ 4비트 양자화보다 메모리 더 많이 사용
```
